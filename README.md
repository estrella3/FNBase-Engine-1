# 개요
[fnbase.xyz](https://fnbase.xyz)에서 배포하는 경량 게시판 엔진으로, PHP를 기반으로 작동합니다.
기본적인 게시판의 기능을 충실하게 제공할 예정입니다.

현재 최신 버전은 0.8.14입니다.

# 요구 사항
항목 | 최소 | 권장 |
---- | ---- | ---- |
PHP | 5.4 + | 7.2 + |
데이터베이스 | MySQL | MariaDB |
웹 서버 |  | Apache |
